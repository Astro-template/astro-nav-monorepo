# Website 包手动验证清单

## 🚀 开始验证

### 步骤 1: 启动开发服务器
```bash
cd astro-nav-monorepo
pnpm --filter @astro-nav/website dev
```

等待服务器启动，应该看到类似以下输出：
```
astro  v5.15.1 ready in XXX ms

┃ Local    http://localhost:4321/
┃ Network  use --host to expose
```
